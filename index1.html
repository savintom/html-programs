HTML means hyper text markup language.
Hyper means link or reference.
Text means information like video, audio, pictures and includes texts as well.
Markup means predefined tag like the paragraph tag or heading tag, etc.
Many of these tags make a language,hence called markup language.

HTML is used to design or make the front end of webpages and websites. Hence, HTML is used for web designing.
